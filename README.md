# Campus-Anomaly-Detection

# Deep Learning Mini Project Series  
### MLP + CNN + Temporal Modeling + Generative Models + Deployment

## Overview
This project is a complete deep learning pipeline developed in multiple stages, progressing from fundamental models to advanced architectures and deployment.

The workflow covers:
- Core classification using MLP and CNN  
- Feature extraction with pretrained CNNs and temporal modeling  
- Representation learning using Autoencoders and GANs  
- End-to-end system integration and deployment  

Results
- CNN outperforms MLP on spatial data due to feature learning
- Pretrained models improve feature quality significantly
- LSTM/GRU capture temporal dependencies effectively
- GAN produces realistic samples with stable training
- End-to-end system successfully deployed
